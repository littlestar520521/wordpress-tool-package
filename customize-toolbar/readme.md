# 右下角工具栏

集成了一些有意思的小功能

## 目前功能

* 更换壁纸为今日Bing美图
* 回到页首
* 开启背景音乐
* 手动切换白天/夜间外观（仅限自用版本）
* 自身可手动收起/拉下

## 文件信息

* 通用版本 `uni`：v2.1
